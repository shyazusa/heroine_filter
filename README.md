# heroine_filter

## Usage

### install sinatra

```
$ gem install sinatra
```
### setup heroine_filter


```
$ git clone git@github.com:shyazusa/heroine_filter.git
$ cd heroine_filter
$ bundle install
$ mkdir public
$ mkdir public/images
$ bundle exec ruby app.rb
```

### Use!

Accsess to `http://localhost:4567/`

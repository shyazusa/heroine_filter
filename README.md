# heroine_filter

## Usage

### install sinatra

```
$ gem install sinatra
```

### setup heroine_filter

```
$ git clone git@github.com:shyazusa/heroine_filter.git
$ cd heroine_filter
$ bundle install
$ mkdir public
$ mkdir public/images
$ mkdir public/fonts
```

#### Add fonts

Place the font you want to use in this project in `public/fonts`.

### Use!

```
$ bundle exec ruby app.rb
```

Accsess to `http://localhost:4567/`
